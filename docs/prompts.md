## Step 1：通义万相图组生成

### 图组全局风格定义

> 在通义万相中创建图组时，首先设定统一的风格描述：

```

画面清透有留白，工笔与写意结合，人物造型端庄典雅，场景开阔有纵深，
光影柔和如绢本设色，16:9电影画幅，画面充满诗意和叙事感
```

## Step 2：即梦图生视频

### 操作步骤

1. 打开 **即梦** → https://jimeng.jianying.com
2. 选择 **图生视频**
3. 上传刚才通义万相生成的每张图
4. **保持默认/自动生成动态**（不需要写额外Prompt，让AI自然理解画面内容）
5. 每张图生成 **1 次**，时长选择 **5 秒**
6. 下载视频放入 `videos/raw/`

### 命名规则

```
图1 → p01  → 下载为 p01.mp4
图2 → p02  → 下载为 p02.mp4
...
图8 → p08  → 下载为 p08.mp4
```
